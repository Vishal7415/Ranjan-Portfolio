# Vercel Deployment Guide

## 🚀 Deploy Your Portfolio on Vercel

### Step 1: Go to Vercel
1. Visit [vercel.com](https://vercel.com)
2. Click **"Sign Up"** or **"Log In"**
3. Choose **"Continue with GitHub"** (recommended)

### Step 2: Import Your Repository
1. Click **"New Project"** 
2. Find and select your `ranjan-portfolio` repository
3. Click **"Import"**

### Step 3: Configure Deployment
1. **Framework Preset**: `Other` (static HTML)
2. **Root Directory**: `./` (leave as default)
3. **Build Settings**: Leave empty (no build needed)
4. **Environment Variables**: Not needed for this project
5. Click **"Deploy"**

### Step 4: Wait for Deployment
- Vercel will automatically detect it's a static site
- Deployment takes 1-2 minutes
- You'll get a URL like: `https://your-project-name.vercel.app`

### Step 5: Custom Domain (Optional)
1. Go to **Project Settings** → **Domains**
2. Add your custom domain if you have one
3. Follow DNS instructions provided

## 🔧 Troubleshooting

If deployment fails:
1. Check that `index.html` is in the root directory
2. Ensure repository is public
3. Try redeploying from Vercel dashboard
