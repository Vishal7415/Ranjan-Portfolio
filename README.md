# Ranjan Vishwakarma - AI & Data Science Portfolio

A modern, animated portfolio website for Ranjan Vishwakarma, showcasing skills, projects, and real-time GitHub data integration.

## Features

- 🎨 **Modern Design** - Beautiful gradient animations and smooth transitions
- 📱 **Responsive** - Works perfectly on all devices
- 🚀 **Real-time GitHub Integration** - Live repository data and statistics
- ⚡ **Fast Loading** - Optimized performance
- 🎯 **SEO Friendly** - Semantic HTML5 structure

## Sections

- **Hero** - Introduction with animated entrance
- **About** - Education and background information
- **Skills** - Technical skills with animated progress bars
- **Projects** - Featured portfolio projects
- **GitHub** - Real-time GitHub repositories and statistics
- **Contact** - Contact form and social links

## GitHub Integration

The portfolio fetches real-time data from GitHub API:
- Repository statistics
- Latest repositories (sorted by stars and recency)
- Programming language detection with color coding
- Auto-refresh every 5 minutes

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **JavaScript** - GitHub API integration and interactions
- **Font Awesome** - Icons
- **GitHub API** - Real-time data fetching

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open `index.html` in your browser or serve it locally:
```bash
python3 -m http.server 8080
```

3. To display your GitHub data:
   - Enter your GitHub username in the GitHub section
   - Click "Fetch Data" to load your repositories

## Deployment

### GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Select `main` branch as source

### Vercel
1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect and deploy the static site
3. Get your live URL instantly

## Customization

- Update personal information in the HTML
- Modify color scheme in CSS variables
- Add/remove projects in the Projects section
- Customize GitHub API settings
